# uuidv4

A simple util to generate UUIDv4 (0.5KB)

## Install

```sh
$ npm install --save @kuoruan/uuidv4
```

Or with Yarn

```sh
$ yarn add @kuoruan/uuidv4
```

## Usage

```javascript
import UUIDv4 from "@kuoruan/uuidv4";

console.log(UUIDv4()); // 367d3b57-0b29-445d-b3bd-a7a488e2b26d
```
